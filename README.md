# India Shawn Website

This is a multi-page website for India Shawn.

## Website Structure

```
india-shawn-website/
├── index.html          # Home/Music page - Main landing page with album grid
├── videos.html         # Videos page - Music videos and visual content
├── shows.html          # Shows page - Tour dates and concert information
├── shop.html           # Shop page - Merchandise and products
└── README.md           # This file
```

## Pages Overview

### index.html (Music/Home)
- Main landing page
- Album grid display
- Interactive lightbox for streaming links
- Mobile-responsive design

### videos.html
- Music videos section
- Full-screen video displays
- Visual content gallery

### shows.html
- Tour dates and shows
- "Coming Soon" section for upcoming events

### shop.html
- Merchandise and products
- "Coming Soon" section for store launch

## Navigation

All pages include:
- Fixed header with logo
- Hamburger menu for mobile
- Desktop navigation bar
- Social media links in footer
- Smooth transitions between pages

## How to Use

1. Open `index.html` in a web browser to start
2. Navigate between pages using the menu
3. All pages are fully responsive (mobile & desktop)
4. No server required - works as static HTML files

## Technical Details

- Pure HTML, CSS, and JavaScript
- No external dependencies
- All images embedded as base64
- Mobile-first responsive design
- Cross-browser compatible

## Features

- ✅ Responsive design (mobile & desktop)
- ✅ Interactive album lightbox
- ✅ Smooth page transitions
- ✅ Fixed navigation header

- ✅ Embedded images (no external resources)
